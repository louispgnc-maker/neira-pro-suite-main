# Configuration du compte denis@neira.test

## Résumé
Configurer denis@neira.test comme compte de test client avec :
- ✅ Accès espace **Avocat**
- ✅ Abonnement **Cabinet-Plus** (illimité)

## Méthode 1 : Via Dashboard Supabase (RECOMMANDÉ)

1. Aller sur https://supabase.com/dashboard/project/elysrdqujzlbvnjfilvh/editor
2. Ouvrir le **SQL Editor**
3. Copier-coller le contenu de `configure-denis-account.sql`
4. Cliquer sur **Run**

## Méthode 2 : Via l'app

1. Se connecter avec votre compte admin
2. Aller dans **Mon cabinet**
3. Inviter **denis@neira.test** comme membre
4. Aller dans **Paramètres** → **Facturation**
5. Mettre à jour vers **Cabinet-Plus**
